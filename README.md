# 🔐 Cyber Security Toolkit (Python)

A Python-based **Cyber Security Toolkit** built using **Tkinter GUI**.  
This project combines multiple security and utility tools into a single application.

---

## 🚀 Features

- 🔑 User Login & Signup System
- 🌐 Domain to IP Finder
- 🔍 SQL Injection Checker
- 🛰 Open Port Scanner
- 📂 File Organizer
- 📸 Screenshot Capture Tool
- 🔐 Password Strength Checker
- 🔑 Random Password Generator
- 💻 System Information Viewer
- 📧 Automatic Email Sender
- 🌍 Subdomain Finder

---

## 🛠 Technologies Used

- Python
- Tkinter (GUI)
- Socket Programming
- Requests Library
- PyAutoGUI
- OS & Shutil Modules
- Platform Module
